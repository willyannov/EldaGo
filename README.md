# EldaGo

Sistema de cardápio digital desenvolvido para a doceria **Elda Bolos e Doces** (Sorocaba/SP).

Projeto Integrador III — **UNIVESP** (Universidade Virtual do Estado de São Paulo).

O sistema permite que os clientes acessem o cardápio pelo celular via QR Code, montem o pedido e acompanhem o status em tempo real, enquanto a atendente gerencia a fila pelo painel administrativo.
